# Laundry Services Web App

This is a simple laundry booking website made for Tutedude Assignment 4.

## Features

- Responsive navbar
- Hero section with scroll button
- Service achievements section
- Services list with add and remove buttons
- Cart total calculation
- Booking form
- EmailJS booking confirmation
- Quality section
- Newsletter form
- Footer section

## Setup Instructions

1. Open the project folder in VS Code.
2. Replace the EmailJS values in `script.js`:
   - `YOUR_PUBLIC_KEY`
   - `YOUR_SERVICE_ID`
   - `YOUR_TEMPLATE_ID`
3. Create an EmailJS account if you do not have one.
4. Create an email service and a template in EmailJS.
5. Copy the credentials into `script.js`.

## How to Run Locally

1. Open `index.html` in a browser.
2. Or use Live Server in VS Code.

## Deployment

This project can be deployed on Netlify by uploading the project folder or connecting the GitHub repository.

## EmailJS Credentials

To get EmailJS credentials:

1. Go to the EmailJS website.
2. Sign up or log in.
3. Add your email service.
4. Create a template.
5. Copy your Public Key, Service ID, and Template ID into `script.js`.
